
# Shop Project

This project uses API to receive data and has different functionalities like wishlist, search, filter, and basic auth.


## Tech Stack

React, Redux Toolkit, Redux Persist, MaterialUI, Formik, Yup  


## Features

- Light/dark mode toggle
- Wishlist
- Search
- Filter
- Basic Auth
- User's Cart
